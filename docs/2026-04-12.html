퀴즈
Q1. AI 시대 보안 담당자가 주로 다루는 두 가지 영역은?
A) 모델 성능 최적화 + 데이터 클리닝
B) AI 사용 보안 + AI 기반 보안
C) GPU 클러스터 관리 + 모델 배포
D) A/B 테스트 설계 + 인과 추론
정답 보기
✓ B) AI 사용 보안 + AI 기반 보안
세션 설명에 명시된 핵심 구분점입니다. AI 사용 자체의 보안 리스크와 AI를 활용한 보안 강화가 주요 주제입니다.
Q2. AI 모델 도입 시 반드시 고려해야 할 보안 리스크는?
A) 모델 해석성 부족
B) 데이터 유출 가능성
C) AUC 점수 저하
D) GPU 사용 비용 증가
정답 보기
✓ B) 데이터 유출 가능성
AI 사용 보안 영역에서 강조된 핵심 리스크로, 데이터 유출과 모델 탈취가 주요 고려사항입니다.
