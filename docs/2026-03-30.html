💡 트랜잭션 외부에서 MySQL 쿼리 병행 처리를 통해 Oracle 쿼리 성능 영향 최소화
퀴즈
Q1. 무중단 마이그레이션 시 Oracle과 MySQL을 동기화하기 위해 사용된 핵심 전략은?
A) A. 데이터베이스 복제
B) B. 이중 쓰기(dual write)
C) C. 캐시 동기화
D) D. 로그 파일 비교
정답 보기
✓ B) B. 이중 쓰기(dual write)
이중 쓰기는 모든 트랜잭션을 두 DB에 동시에 반영해 데이터 정합성을 유지하며 무중단 전환을 가능하게 합니다.
Q2. 트랜잭션 처리 시 MySQL 쿼리 실패를 어떻게 대응했나요?
A) A. 분산 트랜잭션 강제 적용
B) B. 실패 시 자동 롤백
C) C. 트랜잭션 외부에서 병행 처리
D) D. MySQL 쿼리 무시
정답 보기
✓ C) C. 트랜잭션 외부에서 병행 처리
트랜잭션 외부에서 병행 처리를 통해 MySQL 실패가 Oracle 트랜잭션에 영향을 주지 않도록 설계했습니다.
